# Longread_go

Go lecture notes from the Central University backend course. 11 longreads covering the language from basics to HTTP services.

## Contents

| # | Topic |
|---|-------|
| 1 | Introduction to Go — history, philosophy, toolchain |
| 2 | Composite types — arrays, slices, maps, reading Go source code |
| 3 | Functions, structs, methods, interfaces |
| 4 | Generics, error handling, panics |
| 5 | Project structure — packages, modules, dependencies, architecture patterns |
| 6 | Concurrency and parallelism — goroutines, the Go scheduler |
| 7 | Context and sync primitives — `sync`, `WaitGroup`, `Mutex`, concurrency patterns |
| 8 | File I/O operations |
| 9 | Go tooling and codebase practices — linters, formatting, code quality |
| 10 | Config files — JSON and YAML parsing |
| 11 | HTTP and REST — building HTTP services in Go |


## Stack

- Go 1.21+
